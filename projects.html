Big(ger) Projects.

screenfetch-c

screenfetch-c is a rewrite of screenFetch, the popular system information script, in C.
It displays a variety of system statistics next to an ASCII representation of an OS logo.
Compatible with Windows (Cygwin), OS X, Linux, BSD, and Solaris-based systems.

CGoL

CGoL is an implementation of Conway's Game of Life, the cellular automaton, in Java.
It operates on a finite, user-defined n-by-m plane with support for real-time cell insertion and file input.

et

et is a dumb client that connects to a command-and-control IRC server, accepting commands to execute on the local machine
It serves as a proof-of-concept for a simple IRC-based botnet of blind clients.

Complex

Complex is a lightweight implementation of complex numbers in Java.
It provides an object-oriented interface to the real and imaginary components of a complex, as well as a set of common functions for operations on Complex objects.

Fractal

Fractal is a recursive, fractal-generating program that uses Complex and Java's standard imaging libraries to generate Mandelbrot-based fractals on the imaginary plane.

EMG

A group of Arduino Wiring and Processing programs that receive EMG/EKG data from an Olimex shield and interpret it as a proof-of-concept for potential future stress and anxiety therapies.
Developed during the HackBCA hackathon (Top 20 Winner!)

WebService::UrbanDictionary

A Perl module for querying UrbanDictionary.com's JSON API.
Supports retrieving definitions, authors, popularity metrics, and more.

SimpleMail

A Java-based mail client capable of sending mail via SMTP through Yahoo!, Gmail, and Hotmail.
Uses the JavaMail API for transmission.

bbgpio

A C wrapper and library for general purpose I/O (GPIO) on BeagleBones.
Provides convenience functions for opening, reading, and writing data to and from ports.
