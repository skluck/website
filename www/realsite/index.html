Welcome to SCS.

What is this place?

This is SCS, or the website formerly known as superchillspot.com.
It used to be my home page, but I've realized that people want to see a nice entrance rather than a simple index.
It's slowly being phased out, in favor of the new site. To view it, click here.

Why is it so simple?

This website uses pure HTML5 + CSS (with some JS for form validation) for a number of reasons:
For one, it makes it much easier to host, as the server does less work in serving content to you (the user).
It's also more secure (fewer scripts running in the background, no databases to inject).
Finally, I'm not a big fan of dynamic websites when static ones will do just fine.

What can you do on it?

For one, you can check out the services I offer to the public.
Without a shell account, you're free to browse these pages, download some files, and complain to me about how bad it looks.
